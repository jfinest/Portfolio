
Overview
	
Currently this is an updated draft of a portfolio site that has be updated with actual projects that have been completed in the Udacity course. A Github link has ben provided, once clicked you will be re-directed into Github page to view code written for each project.

Download Portfolio

Instructions

1. You will find a zipped file named Portfolio
2. Using your prefer program unzip file
3. In your current browser go to File, Open
4. Browse to the path location where Portfolio has been downloaded into
5. Once open the browser will display portfolio
